#インストール方法 
1.リポジトリをクローン
git clone 
cd text-edit-sample
2.依存パッケージをインストール
npm install
3.環境変数を設定。.envファイルを作成して、そこに以下の例を参考にしてデータベースURLを記述してください。
DATABASE_URL="postgresql://your_name:your_password@localhost:5432/your_database"
4.開発サーバーを起動
npm run dev
